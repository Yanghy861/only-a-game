# 垂钓纪元

一个网页挂机 RPG 原型，方向是多技能成长、资源互锁、装备制作、区域战斗与长期收集。

## 运行方式

需要通过本地 HTTP 服务运行，不能直接用 `file://` 打开。

```bash
python -m http.server 8027
```

然后访问：

```text
http://127.0.0.1:8027/
```

## 项目结构

```text
config/                 游戏配置和数值
css/                    样式
js/                     游戏逻辑和页面脚本
index.html              主页面
项目技术方案.md          后续开发的技术基准
后续开发文档.md          产品方向和长期规划
战斗系统设计文档.md      战斗玩法设计
```

## 后续开发约定

新增玩法或重构时优先参考：

```text
项目技术方案.md
```

核心原则：

- 新玩法不要直接塞进 `main.js`。
- 采用 `config + state + UI + game_systems` 的结构。
- 修改后做 JS 语法检查和浏览器验证。
