# 战斗系统设计文档

## 目标定位

战斗系统的核心不是手动点技能,而是作为挂机游戏的世界推进器。

最终方向:

```text
自动刷怪 + 自由装备构筑 + 探索解锁
```

玩家在战斗外做决策:选择出击方式、选择区域、调整装备、观察收益。战斗过程自动进行。

## 核心原则

1. 不做战斗中手动操作。
2. 装备不绑定近战/远程/魔法,只提供属性和效果。
3. 玩家选择近战/远程/魔法作为出击方式,由出击方式决定经验成长和基础战斗倾向。
4. 内容不一次性摊开,通过已发现、传闻、未知来制造探索感。
5. 怪物掉落要进入装备制作,让战斗和生活技能互相咬合。

## 最小可行版本

| 项目 | 设计 |
|---|---|
| 战斗方式 | 近战、远程、魔法 |
| 区域数量 | 3 个区域 |
| 每个区域 | 2-3 个普通怪 |
| 每个区域核心 | 1 个关键威胁 |
| 区域显示 | 已发现区域、当前威胁、传闻、未知 |
| 普通怪战斗 | 自动循环挂机 |
| 掉落用途 | 怪物材料进入战斗装备制作 |
| 手动技能 | 不作为核心玩法,后续应移除旧入口 |

## 战斗方式

| 方式 | 作用 | 说明 |
|---|---|---|
| 近战 | 稳定推进 | 偏生命和防御,适合稳妥刷怪 |
| 远程 | 快速刷怪 | 偏速度和暴击,适合效率刷材料 |
| 魔法 | 爆发解法 | 偏攻击和暴伤,更脆但爆发高 |

注意:装备不限制出击方式。玩家可以穿高防装备用魔法,也可以穿速度装备打近战。

## 装备槽位

| 槽位 | 说明 |
|---|---|
| 武器 | 单手武器、双手武器、弓、法杖等 |
| 副手 | 盾牌、副手武器、法器、箭袋等 |
| 上衣 | 胸甲、皮甲、长袍等 |
| 下装 | 腿甲、长裤、袍裤等 |
| 鞋子 | 靴子、轻靴、法靴等 |
| 手臂 | 手套、护腕、臂甲等 |
| 护肩 | 肩甲、披肩、护肩等 |
| 戒指 | 暴击、掉落、特殊属性等 |
| 项链 | 生存、抗性、核心构筑等 |

武器规则:

| 类型 | 规则 |
|---|---|
| 单手武器 | 占武器槽,可搭配副手 |
| 双手武器 | 占武器槽,同时锁定副手 |
| 副手装备 | 如果当前武器是双手,不能装备副手 |

## 初始三个区域

| 区域 | 普通怪 | 关键威胁 | 主要掉落 | 解锁方向 |
|---|---|---|---|---|
| 林地边缘 | 史莱姆、野狼、树精残枝 | 林地守卫 | 凝胶、皮革、树脂 | 矿洞传闻、基础装备 |
| 矿洞入口 | 哥布林、矿洞蝙蝠、石壳虫 | 矿洞监工 | 铁片、蝙蝠翼、粗矿粉 | 金属装备、潮湿洞穴传闻 |
| 潮湿洞穴 | 毒蛛、洞穴蜥蜴、孢子兽 | 沼泽毒蛛 | 毒囊、蛛丝、湿鳞 | 毒抗、解毒食物、后续区域 |

## 页面结构

```text
战斗 Lv.X        [近战] [远程] [魔法]

左侧:探索
  已发现区域
  当前威胁
  传闻
  未知

中间:自动战斗
  当前区域
  当前怪物
  击杀进度
  开始/暂停
  掉落预览

右侧:构筑与收益
  当前出击方式
  装备摘要
  主要属性
  最近日志
```

## 核心循环

```text
生活技能生产基础资源
↓
制作装备/宝石/卡片
↓
选择出击方式和战斗区域
↓
自动刷普通怪获得经验、金币、怪物材料
↓
用材料制作新装备
↓
击败关键威胁
↓
解锁新区域、新传闻、新配方
```

## 后续扩展方向

1. 装备特殊效果:吸血、反击、穿透、掉落加成、元素加成。
2. 区域危险度:低危、高危、需要抗性。
3. 自动食物:烹饪系统进入战斗补给链。
4. 宝石镶嵌:让宝石系统影响构筑。
5. 图鉴区域化:只显示当前区域发现度,隐藏总量。
6. 关键威胁机制化:不同威胁要求不同构筑解法。
